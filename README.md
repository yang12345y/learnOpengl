1. 使用反射率方程模拟光的视觉效果。
  (1) 微平面模型。粗糙度：在所有平面中，有多少比例的微平面法线能对齐h(半程向量)，越多，则镜面高光更锐利，范围更小。
  (2) 能量守恒。当一束光碰撞到一个表面的时候，分离成一个反射部分(镜面光照Ks)和一个折射部分(漫反射光照Kd)。
  (3) BRDF。本质是从wi方向入射的光有多少比例会反射到wo。
  (4) 不同材质对反射、折射的分配规则完全不同。
  (5) 反射率方程汇总半球内的所有入射光，经过材质反射后，在观察方向上的总亮度。 
2. 使用BRDF双向反射函数，基于微平面理论近似求得材质的反射和折射属性。描述入射光被表面反射到观察方向的比例。
  (1) 菲涅尔方程，当垂直观察的时候，任何物体或材质表面都有一个基础反射率F0。
3. 使用HDR平均地将所有亮度值分散到LDR，使用Reinhard映射算法。
4. 在颜色最终显示到屏幕前使用gamma矫正。
5. metalness是金属度，用来描述一个材质是金属还是非金属，金属Ks=1，Kd=0，无漫反射。
